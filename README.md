# nuqs-storybook-adapter

Nuqs v2 adapter compatible with @storybook/nextjs-vite mock router

## Installation

```bash
bun add nuqs-storybook-adapter
```

## Usage

```typescript
import { greet } from "nuqs-storybook-adapter";

console.log(greet("World")); // Hello, World!
```
